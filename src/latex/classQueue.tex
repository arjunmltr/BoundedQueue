\subsubsection*{template$<$typename T$>$\newline
class Queue$<$ T $>$}

A thread-\/safe queue implementation with a maximum size. 

This class provides a thread-\/safe queue with a maximum size. It supports push and pop operations and ensures thread safety using mutex and condition variables.
